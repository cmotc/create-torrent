instaTorrent
This tool helps you generate web-seeded torrents with ease, thanks to feross/create-torrent and substack/node-browserify.
It's basically a static web site(actually it's definitely a static web site) that loads the create-torrent library used by the wonderful feross/webtorrent application and gives you a simple way to use it to create a torrent for a file which is already on the web(by using web seeding).
